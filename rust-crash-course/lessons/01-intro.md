# Rust Crash Course — Introduction

Welcome! This repository is an opinionated, practical, and hands-on Rust
crash course. Each lesson contains concise definitions, examples, and
exercises you can run locally with `cargo`.

Goals:
- Explain core Rust concepts clearly and practically.
- Provide runnable examples and small exercises with solutions.
- Demonstrate idiomatic Rust and common pitfalls.

How to use this repo:
- Read lessons in order.
- Try the examples with `cargo run --example <name>`.
- Do exercises in `exercises/` and verify with `cargo test`.

Stay curious and iterate — Rust rewards practice.
